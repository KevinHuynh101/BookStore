<h2 align="center">Run Locally </h2>
Clone the project

```bash
git clone https://github.com/KevinHuynh101/BookStore.git
```

Install a package

```bash
npm i
```

Edit connect MongoDB in file app.js

```bash
mongoose.connect("mongodb://127.0.0.1:27017/bookstore");
```
Run 

```bash
npm start
```
